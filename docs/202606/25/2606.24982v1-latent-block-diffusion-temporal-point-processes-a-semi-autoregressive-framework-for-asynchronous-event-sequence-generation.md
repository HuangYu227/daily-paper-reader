## 摘要
对异步事件序列的潜在分布进行建模和采样在包括社交网络、医疗诊断和金融交易在内的各种实际应用中至关重要。现有的自回归方法在多步生成过程中存在误差累积问题，而非自回归扩散方法通常局限于固定长度的输出序列。在本文中，我们提出了潜在块扩散时间点过程（LBDTPP），一种新颖的半自回归TPP框架，该框架引入潜在块扩散机制，用于高质量且可变长度的事件序列生成。其核心思想是在潜在空间中对事件块定义自回归概率分布，并在每个块内进行高斯扩散。通过顺序生成块的同时在每个块内并行采样事件，LBDTPP保留了自回归TPP的长度灵活性，并继承了扩散模型的高质量并行生成能力。理论上，我们推导了Wasserstein误差界，表明在合适的局部近似和前缀稳定性假设下，块级生成相比事件级自回归生成可以减少误差累积。在六个真实世界基准数据集上的大量实验表明，LBDTPP在无条件生成和条件生成任务中均优于最先进的TPP基线。进一步的实证分析验证了潜在空间扩散和块级生成的益处，并揭示了生成质量与块大小之间的权衡。我们的代码可在https://github.com/Zh-Shuai/LBDTPP获取。

## Abstract
Modeling and sampling from the underlying distribution of asynchronous event sequences are crucial in various real-world applications, including social networks, medical diagnosis, and financial transactions. Existing autoregressive methods suffer from error accumulation during multi-step generation, while non-autoregressive diffusion methods are typically limited to fixed-length output sequences. In this paper, we propose Latent Block-Diffusion Temporal Point Processes (LBDTPP), a novel semi-autoregressive TPP framework that introduces a latent block diffusion mechanism for high-quality and variable-length event sequence generation. The core idea is to define an autoregressive probability distribution over event blocks in latent space and perform Gaussian diffusion within each block. By sequentially generating blocks while simultaneously sampling events in each block, LBDTPP preserves the length flexibility of autoregressive TPPs and inherits the parallel high-quality generation capability of diffusion models. Theoretically, we derive Wasserstein error bounds showing that, under suitable local approximation and prefix-stability assumptions, block-wise generation can reduce error accumulation compared with event-wise autoregressive generation. Extensive experiments on six real-world benchmark datasets demonstrate that LBDTPP outperforms state-of-the-art TPP baselines in both unconditional and conditional generation tasks. Further empirical analyses verify the benefits of latent-space diffusion and block-wise generation, and reveal the trade-off between generation quality and block size. Our code is available at https://github.com/Zh-Shuai/LBDTPP.

- **核心问题**：如何同时实现**变长生成**（避免固定长度限制）和**高质量并行生成**（减轻误差累积），且不牺牲生成效率与灵活性。
- **整体含义**：提出一种半自回归框架 LBDTPP，融合自回归和扩散模型的优势，并首次将**块级扩散**和**潜空间建模**结合用于事件序列生成，理论上证明块级生成可减少前缀级误差累积，实验上在六个真实数据集上取得最优性能。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **半自回归范式**：将事件序列在潜空间划分为**块（block）**，块之间采用**自回归**方式（顺序生成，依赖历史块）；块内采用**高斯扩散**并行生成多个事件。这样既保留了自回归 TPP 的变长灵活性，又继承了扩散模型的高质量并行生成能力，同时通过缩短递归步数来减轻误差累积。

### 关键技术细节
1. **事件编码（Event Encoder）**：
   - 将每个事件 \(x_\ell = (\tau_\ell, m_\ell)\) 映射到连续潜向量 \(z_\ell \in \mathbb{R}^D\)：时间嵌入（正弦位置编码）和标记嵌入（线性变换 one-hot 表示）相加。编码器不含可学习参数。
2. **潜在块扩散（Latent Block Diffusion）**：
   - 将长度为 \(L\) 的序列分为 \(B = L/L'\) 个非重叠块（每个块长 \(L'\)）。
   - 概率分解：\(\log p_\theta(z) = \sum_{b=1}^B \log p_\theta(z^b \mid z^{<b})\)，即块间自回归。
   - 每个块的条件分布 \(p_\theta(z^b \mid z^{<b})\) 通过高斯扩散建模：前向加噪，反向去噪（以历史块 \(z^{<b}\) 为条件）。
   - 训练损失简化为预测干净潜块的均方误差：\(\mathcal{L}_{LBD}(z;\theta) = \frac{1}{L} \sum_{b=1}^B \mathbb{E}_{k, z^b, \epsilon^b} \left[ \| z^b - \hat{z}_\theta(z^b_k, z^{<b}, k) \|^2 \right]\)。
   - 使用**块扩散注意力掩码**（Block Diffusion Attention Mask）实现单个前向传播计算所有块的扩散损失，结合 KV 缓存加速训练和推理。
3. **事件解码器（Event Decoder）**：
   - 从潜向量 \(z^\ell\) 重建事件：时间 \(\hat{\tau}_\ell = \text{Softplus}(\text{MLP}_\tau(z^\ell))\)，标记 \(\hat{m}_\ell = \arg\max \text{Softmax}(\text{MLP}_m(z^\ell))\)。
   - 重建损失为时间 MSE 与标记交叉熵之和：\(\mathcal{L}_{Recon}(x,\hat{x};\phi) = \frac{1}{L}\sum_b \sum_{\ell} [(\tau_\ell - \hat{\tau}_\ell)^2 - \log \hat{p}_\ell[m_\ell]]\)。
4. **端到端训练**：
   - 总损失：\(\mathcal{L}_{Overall} = \mathcal{L}_{LBD} + \lambda \mathcal{L}_{Recon}\)，\(\lambda\) 平衡二者（实验中固定 \(\lambda=1\)）。
5. **生成过程**（无条件/条件）：
   - **无条件生成**：从噪声开始，逐块采样潜块，通过扩散反向过程获得潜块 \(z^b\)，用 KV 缓存加速。解码得事件块，重复直至达到终止时间 \(T\)，然后截断。块内并行生成 \(L'\) 个事件。
   - **条件生成**：给定历史事件 \(x_c\)，仅生成未来部分，块分解仅作用于未来潜块，历史潜块作为固定条件。
6. **理论分析**：
   - 在**局部近似误差**和**前缀稳定性**假设下，证明了事件级自回归的 Wasserstein 误差上界为 \(\varepsilon_{AR} A_L(\rho_{AR})\)，块级为 \(\varepsilon_{BL} A_B(\rho_{BL})\)（其中 \(A_n(\rho) = ((1+\rho)^n -1)/\rho\)）。
   - 当 \(\varepsilon_{BL} \le L'\varepsilon_{AR}\) 且 \(\rho_{BL} \le \rho_{AR}=\rho\) 时，块级误差累积因子不超过事件级因子，且 \(\rho>0, L'>1\) 时严格更小。这解释了块大小 \(L'\) 存在最优值的现象。
