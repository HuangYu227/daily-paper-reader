## 摘要
时序因果表征学习在离散时间潜在因果模型中已建立了强大的可识别性结果，但连续时间潜在随机微分方程（SDE）模型的可识别性仍基本开放。我们利用环境诱导的扩散协方差位移来解决这一差距。我们研究通过未知非线性微分同胚观测的加性噪声潜在SDE，具有共享漂移但特定于环境的扩散协方差。我们证明了两个对角扩散机制，其坐标方差比两两不同，可以在没有漂移稀疏性假设的情况下，将潜在坐标识别到置换和缩放。我们首先证明线性奥恩斯坦-乌伦贝克系统的这一结果，然后将其扩展到一般加性噪声潜在SDE。在适度光滑性下，瞬时漂移-雅可比因果图可以识别到相同的置换。我们提出了一种两阶段估计器，用于潜在解缠和可选图恢复；合成系统上的实验确认了预测的可识别性边界，对哈当厄尔大桥监测数据的应用展示了该方法在实际传感器轨迹上的效果。

## Abstract
Causal representation learning for time series has developed strong identifiability results in discrete-time latent causal models, but identifiability in continuous-time latent stochastic differential equation (SDE) models remains largely open. We address this gap using environment-induced shifts in diffusion covariance. We study additive-noise latent SDEs observed through an unknown nonlinear diffeomorphism, with shared drift but environment-specific diffusion covariance. We show that two diagonal diffusion regimes with pairwise distinct coordinate-wise variance ratios identify the latent coordinates up to permutation and scaling, without any sparsity assumption on the drift. We first prove this result for linear Ornstein--Uhlenbeck systems and then extend it to general additive-noise latent SDEs. Under mild smoothness, the instantaneous drift-Jacobian causal graph is identifiable up to the same permutation. We propose a two-stage estimator for latent disentanglement and optional graph recovery; experiments on synthetic systems confirm the predicted identifiability boundary, and an application to Hardanger Bridge monitoring data illustrates the approach on real sensor trajectories.

- **研究动机**：因果表征学习（CRL）在离散时间潜在因果模型中已取得强可识别性成果，但在连续时间潜在随机微分方程（SDE）模型中可识别性仍基本空白。许多科学和工程系统（如气候、生物、健康、经济）在连续时间内演化，且观测变量（如视频帧、传感器流）往往是潜在动态因素的非线性混合。若无额外结构，这些因素无法仅从观测中识别。
- **核心挑战**：连续时间设置下的主要困难在于扩散协方差在非线性坐标变换下不是不变的：在观测坐标系中，局部二次变分是潜在协方差经未知混合的雅可比变换后的结果。单个扩散机制无法确定潜在因果坐标。作者的**关键洞察**是：两个具有逐坐标不同方差比的对角扩散机制同时对同一未知雅可比矩阵施加约束，从而将剩余歧义缩减为置换和缩放。
- **整体含义**：该工作首次给出连续时间潜在SDE状态坐标的可识别性定理级别证明，表明环境诱导的扩散偏移可作为强大侧信息，在不依赖漂移稀疏性的前提下解开非线性混合，恢复潜在坐标和瞬时因果图。

## 2. 方法论：核心思想、关键技术细节、算法流程

- **核心思想**：利用两个环境共享潜在漂移但具有不同对角扩散协方差这一设置。两个环境中的扩散方差比（Σ₁⁻¹Σ₂）的对角元素如果两两不同（简单谱条件），则能唯一固定潜在坐标系（至多置换和缩放）。然后通过瞬时漂移-雅可比图给出潜在因果结构。
- **技术细节**：
  - **模型设定**：潜在状态 $z_t^{(e)}\in\mathbb{R}^d$ 服从加性噪声SDE：$dz_t^{(e)} = f(z_t^{(e)})dt + G_e dW_t^{(e)}$，观测 $x_t^{(e)} = g(z_t^{(e)})$。两个环境共享漂移 $f$，仅扩散矩阵 $G_e$ 不同。
  - **关键假设**：
    - (S1) $g,\tilde{g}$ 是 $\mathcal{C}^2$ 微分同胚；$\varphi = \tilde{g}^{-1}\circ g$。
    - (S2) 两个参数化下的观测轨迹定律相等。
    - (S3) 扩散协方差 $\Sigma_e = G_eG_e^\top$ 为对角正定。
    - (S4) 两环境扩散方差比 $\sigma_{2,i}^2/\sigma_{1,i}^2$ 两两不同。
    - (S5) （非线性SDE附加）漂移连续，SDE弱适定。
  - **证明思路**：
    1. 通过二次变分比较，获得沿轨迹的恒等式 $D\varphi(z_t^{(e)})\Sigma_e D\varphi(z_t^{(e)})^\top = \tilde{\Sigma}_e$。
    2. 利用开集正性引理（由非退化加性噪声保证状态在任意时间具有全支撑分布）将沿路径等式提升为逐点恒等式。
    3. 用环境1白化得到正交矩阵场 $M(z) = \tilde{\Sigma}_1^{-1/2} D\varphi(z) \Sigma_1^{1/2}$。
    4. 环境2给出 $M(z) K M(z)^\top = \tilde{K}$，$K$ 有简单谱，由正交矩阵对角化推出 $M(z)$ 是带符号的置换矩阵，且由连续性得常值。
    5. 因此 $D\varphi(z) \equiv L$ 为常值单项矩阵，$\varphi(z) = Lz + c$ 为仿射变换。
    6. 代入漂移关系得 $\tilde{f}(Lz+c) = Lf(z)$，进而确认漂移和因果图的可识别性。
- **算法流程（两阶段估计器）**：
  - **阶段1：短时转移拟合**。在编码空间中使用欧拉-丸山离散近似：$\tilde{z}_{t+\Delta t} \mid \tilde{z}_t, e \sim \mathcal{N}(\tilde{z}_t + \tilde{f}_\psi(\tilde{z}_t)\Delta t, \tilde{\Sigma}_e \Delta t)$。通过最小化包含雅可比行列式项的负对数似然 $L_{S1}$ 联合学习编码器 $h_\theta$、漂移模型 $\tilde{f}_\psi$ 和扩散参数 $\tilde{\sigma}_{e,i}^2$。
  - **阶段2（可选）：稀疏漂移回归**。固定编码器，形成速度目标 $\hat{v}_t = (h_\theta(x_{t+s\Delta t}) - h_\theta(x_t))/(s\Delta t)$，并最小化 $L_{S2}$：速度MSE + 漂移-雅可比 $\ell_1$ 惩罚。最后通过阈值化提取因果图。
